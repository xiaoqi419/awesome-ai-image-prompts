# 🎨 Awesome GPT Image 2 - GPT Image 2 提示词精选合集

**🔗 仓库链接**: https://github.com/ZeroLu/awesome-gpt-image

## 项目简介

**awesome-gpt-image** 是一个 GPT Image 2 提示词（prompt）精选合集，收录了来自 X (Twitter) 社区的高质量 GPT Image 2 使用示例。

GPT Image 2 是 OpenAI 最新发布的图像生成模型，相比 1.5 版本有显著提升：
- ✅ 更好的世界知识 - 更准确地呈现真实物体和场景
- ✅ 更强的风格理解 - 可复制特定艺术风格和游戏美学
- ✅ 更好的提示词遵循 - 精确执行复杂指令
- ✅ 照片级真实输出 - 生成看起来像真实照片或截图的图像

## 精选示例

### 🎮 游戏风格生成

**GTA 6 游戏画面风格**
```prompt
GTA 6 in-game footage, very detailed, very realistic. Close-up shot taken from a stationary 4k monitor.
```

**Zelda: Tears of the Kingdom 风格**
```prompt
In the game Zelda totk link is in a e531 series train made by him
```

### 📷 真实摄影

**RAW iPhone 质量 - 地铁站**
```prompt
Create a completely RAW quality, unprocessed, unedited image with full iPhone camera quality. A subway station in USA.
```

**YouTube 视频截图风格**
```prompt
Screenshot of a YouTube video showing someone who time-traveled to the Middle Ages
```

## 仓库特色

- 📊 按使用场景分类（游戏、摄影、视频、社交媒体）
- 🖼️ 每个 prompt 都附带实际效果对比图
- 🔄 包含模型对比（GPT Image 1.5 vs 2、Nano Banana vs GPT）
- 📈 包含 Star History 图表
- 🌐 所有图片使用 CDN，加载快速

## 使用场景

1. **学习提示词工程** - 了解如何编写高效的图像生成 prompt
2. **获取创作灵感** - 参考现有 prompt 进行二次创作
3. **模型对比研究** - 理解不同 AI 图像模型的特点
4. **快速上手 GPT Image 2** - 直接使用经过验证的 prompt
